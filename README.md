# Smart-Stats-Analyzer
This project is a Python-based Data Analysis Tool that allows users to input numerical data and instantly perform statistical operations using powerful libraries like NumPy and pandas.
The program demonstrates the practical use of data science libraries in a simple command-line interface. It calculates key statistical measures such as mean, median, mode, standard deviation, variance, minimum, maximum, and sum. Additionally, it uses pandas to organize the data into a structured DataFrame, generate summary statistics, analyze frequency distribution, and create new computed columns.

This project is ideal for beginners who want to understand how NumPy and pandas work together for data analysis tasks.

Features:
Accepts user input dynamically
Performs statistical analysis using NumPy
__Calculates:
Mean, Median, Mode
Standard Deviation & Variance
Min, Max, and Sum
__Uses pandas to:
Create DataFrame
Generate summary statistics
Display value counts (frequency)
Add computed columns (e.g., squared values)
Beginner-friendly and easy to run
